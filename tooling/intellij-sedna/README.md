# IntelliJ SEDNA plugin (deferred)

**Current version limitation:** a full IntelliJ plugin is not part of v1.x. It will be considered only after core Java/Spring DNA functionality is complete.

Phase 14 DX visualization is delivered via CLI:

```bash
sedna visualize --input=project.sdna --output=graph.dot
```

Import the DOT file in IntelliJ (Graph support plugin) or render with Graphviz.
